## 🚀 Project Overview

**SAMS (Student Attendance Management System)** is a desktop-based system built using **JavaFX and MySQL**.  
It helps educational institutes automate attendance management and reduce manual work.

The system includes a **secure role-based login system** for Admin and Lecturer users.

---

## ✨ Key Features

### 👨‍💼 Admin Panel
- 📚 Manage Courses  
- 🎓 Manage Students  
- 👨‍🏫 Manage Lecturers  
- 📖 Manage Subjects  
- 📅 Class Scheduling System  
- 📊 Attendance Tracking  
- 📑 Generate Reports  
- 🔐 Secure Authentication  

---

### 👨‍🏫 Lecturer Panel
- 📅 View Assigned Classes  
- 📝 Mark Student Attendance  
- 📊 View Attendance Reports
